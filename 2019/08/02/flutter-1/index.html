Flutter 入坑记（一：环境搭建）

发表于 2019-08-02 | 分类于 flutter | | 阅读次数
字数统计 615 | 阅读时长 3

Flutter是谷歌的移动UI框架，可以快速在iOS和Android上构建高质量的原生用户界面。 Flutter可以与现有的代码一起工作。在全世界，Flutter正在被越来越多的开发者和组织使用，并且Flutter是完全免费、开源的。

安装

这里以 win10 系统为例

下载 Flutter 安装包

配置环境变量

控制面板>用户帐户>用户帐户>更改我的环境变量

变量名 变量值
PUB_HOSTED_URL https://pub.flutter-io.cn
FLUTTER_STORAGE_BASE_URL https://storage.flutter-io.cn
Path 追加变量 安装包路径\flutter\bin

安装依赖

检测是否需要安装任何依赖项来完成安装

根据提示进行处理

  1. 安装 Android Studio

提示的下载路径需要翻墙，这里提供一个不用翻墙的下载路径 Android studio中文网

下载好后打开直接点击下一步安装

点击 cancel

安装好后，进入 Android studio ，点击 Plugins ,搜索 Flutter 进行安装

  1. 安装 Flutter extension

访问 Flutter extension 进行安装

安装完成后再运行 flutter doctor 进行检测

按提示执行命令，然后一直输入 y 即可

再运行 flutter doctor 检测

创建应用

重新打开 Android studio ，选择 Start a new Flutter project

设置项目名称和存储位置

运行应用

配置模拟器

选择一个模拟器设备

选择一个镜像，下载

启动应用（首次启动时间较长）

报错记录

  1. 命令行执行 flutter doctor
    报错：

解决方法:
删除下载的Flutter安装包，执行以下命令获取安装包

  1. Android studio 安装报错

原因：win10 系统启用了 Hyper-V
解决方法：

  • 管理员身份运行cmd
  • 执行 bcdedit /enum

  • 执行 bcdedit /set {current} hypervisorlaunchtype off
  • 电脑重启
  1. Android studio 点击 run 时报错

原因：镜像版本过高
解决方法：选择低版本的镜像
